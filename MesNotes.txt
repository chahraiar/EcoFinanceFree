Pour réparer ton projet proprement :

Supprime les modules et le cache :
    Remove-Item -Recurse -Force node_modules
    Remove-Item -Force package-lock.json
    npm cache clean --force

Supprimez le dossier .expo :
    Remove-Item -Recurse -Force .expo

Installe les dépendances sans conflit :
    npm install --legacy-peer-deps
ou :
    npm install --force <<----------------------------

Lancez l'application avec :
    
Si des erreurs persistent Vérifiez la configuration:
    npx expo doctor --fix

    npm install --legacy-peer-deps expo-font@~11.4.0
    npm install expo-font@11.3.0 --legacy-peer-deps
